Track A
Robot Software
  • C++ (C++14/17+) MUST— Performance-critical robot control nodes, real-time components
  • Python MUST— Prototyping, data pipelines, test harnesses, orchestration scripts
  • ROS 2 MUST— Nodes, Topics, Services, Actions, QoS. Skip ROS 1 (end-of-life)
  • Linux HIGH— Nearly all robots run Linux; field debugging depends on Linux fluency
  • Git / Docker HIGH— Version control and containerized development/deployment
  • Gazebo / MATLAB MID— Simulation environments for testing before hardware deployment
Track B
Robot Hardware & Mechanical
  • CAD (SolidWorks, CATIA) MUST— 3D modeling of frames, joints, enclosures, linkages
  • PCB / Circuit Design HIGH— Schematic capture, power supply design, sensor interface circuits
  • Motor & Actuator Integration HIGH— DC, servo, stepper motors; BLDC controllers; DYNAMIXEL actuators
  • Embedded (Arduino, STM32, RPi) HIGH— MCU firmware, real-time control loops, sensor driver development
  • PLC Programming MID— Ladder logic for industrial automation and factory robotics lines
Track C
AI, Perception & Autonomy
  • Computer Vision MUST— OpenCV, object detection, image segmentation, depth estimation
  • SLAM HIGH— Simultaneous Localization and Mapping using LiDAR, cameras, IMU fusion
  • Deep Learning (PyTorch) HIGH— Object recognition, imitation learning, reinforcement learning for control
  • Path Planning & Navigation HIGH— A*, RRT, potential fields, Nav2 stack in ROS 2
  • Sensor Fusion MID— Kalman filters, multi-sensor integration (LiDAR + camera + IMU)
Track D
Control Theory & Math Foundations
  • Linear Algebra MUST— Transforms, rotations, homogeneous coordinates — daily use in robotics
  • Control Theory (PID+) HIGH— PID tuning, state-space control, real-time feedback systems
  • Kinematics & Dynamics HIGH— Forward/inverse kinematics, multi-joint robot arm dynamics
  • Probability & Statistics MID— Bayesian filtering, noise modeling, uncertainty estimation
  • Optimization MID— Trajectory optimization, convex optimization for motion planning
Education & Degrees
  • Bachelor's in ME, EE, CS, or Robotics — minimum for most positions
  • Master's preferred for R&D roles (control, AI, perception) at large corps
  • Project portfolio or competition experience can offset degree level at startups
  • KAIST, POSTECH, SNU, Hanyang robotics labs are strong pipelines to industry
Certifications & Extras
  • Robot competitions (RoboCup, DARPA, KRoC) — valued as hands-on proof
  • Open-source ROS 2 contributions — strong differentiator for SW track
  • Personal robot projects on GitHub — show initiative and practical skill
  • Korean certifications: 로봇SW개발기사, 전기기사, 정보처리기사
  • English communication (technical reading/writing) — expected at most companies
